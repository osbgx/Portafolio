## Resumen
Desarrollé el frontend completo del sitio web **[doctorfy.org](https://doctorfy.org)**, una plataforma orientada a servicios de salud digital. El proyecto se construyó utilizando un template de **shadcn** adaptado a **Nuxt.js**, aprovechando componentes modernos, diseño minimalista y una arquitectura modular basada en Vue y Tailwind CSS.

El objetivo fue entregar una landing profesional, rápida y escalable, con páginas internas informativas y una experiencia visual coherente con la identidad de Doctorfy.

## Alcance del proyecto
El trabajo se centró exclusivamente en el **diseño y desarrollo frontend**, sin backend ni integraciones complejas. Las principales entregas incluyeron:

- Landing page principal.
- Páginas internas informativas (servicios, contacto, beneficios, etc.).
- Componentes reutilizables basados en shadcn.
- Navegación responsiva y optimizada.
- SEO básico (estructura semántica, metadatos, performance inicial).
- Adaptación del template a la identidad visual del proyecto.

## Tecnologías utilizadas
El desarrollo se realizó utilizando:

- **Nuxt.js** como framework principal.
- **Vue 3** para la estructura de componentes.
- **Tailwind CSS** para estilos utilitarios.
- **shadcn** como base de componentes UI.
- **JavaScript** para interactividad ligera.
- **TypeScript** para tipado y mayor seguridad.
- **Nuxt Content** para gestión de contenido estático.
- **Nuxt UI** para componentes adicionales y mejoras de UX.
- **Nuxt Image** para optimización de imágenes.
- **Nuxt Icon** para iconos consistentes.
- **Nuxt SEO** para optimización de SEO básico.
- Adaptación de un template existente, reorganizado y personalizado según las necesidades del cliente.

## Diseño y experiencia de usuario
El enfoque estuvo orientado a:

- Interfaz moderna y minimalista.
- Componentes visualmente consistentes gracias a shadcn.
- Navegación clara y fluida.
- Adaptabilidad total a dispositivos móviles.
- Uso de layouts y secciones modulares para facilitar futuras expansiones.

Se trabajó sobre un template shadcn, pero se realizaron personalizaciones profundas en estilos, estructura y componentes para ajustarlo a la identidad visual de Doctorfy.

## Resultado
El sitio final entregado fue:

- Moderno, rápido y completamente responsive.
- Basado en componentes reutilizables y mantenibles.
- Visualmente consistente con la marca.
- Preparado para futuras expansiones, como dashboards, login o funcionalidades dinámicas.

El proyecto cumplió con los requerimientos del cliente y dejó una base sólida para el crecimiento de la plataforma.
