## Resumen
Desarrollé el frontend completo del sitio web **[saludy.com.py](https://saludy.com.py)**, una plataforma institucional orientada a servicios de salud y medicina prepaga. El proyecto consistió en construir una landing moderna y varias páginas internas utilizando una plantilla base de Bootstrap, adaptada y personalizada mediante HTML, CSS, JavaScript y Sass.

El objetivo fue entregar un sitio claro, accesible y visualmente consistente, con una estructura modular que permitiera futuras expansiones y rediseños.

## Alcance del proyecto
El trabajo se centró exclusivamente en el **diseño y desarrollo frontend**, sin backend ni integraciones complejas. Las principales entregas incluyeron:

- Landing page principal.
- Página de contacto.
- Página de prestadores.
- Página de planes y servicios.
- Secciones informativas adicionales según requerimientos del cliente.
- Navegación responsive y optimizada.
- SEO básico (metadatos, estructura semántica, performance inicial).

## Tecnologías utilizadas
El desarrollo se realizó utilizando:

- **HTML5** para la estructura del sitio.
- **CSS3 y Sass** para estilos personalizados y mantenibles.
- **JavaScript** para interactividad ligera.
- **Bootstrap** como base de diseño responsive.
- Adaptación de una plantilla existente, optimizada y reorganizada según las necesidades del cliente.

## Diseño y experiencia de usuario
El enfoque estuvo orientado a:

- Claridad visual.
- Jerarquía de información.
- Navegación simple y directa.
- Adaptabilidad a dispositivos móviles.
- Uso de componentes reutilizables para mantener consistencia.

Se trabajó sobre una plantilla Bootstrap, pero se realizaron personalizaciones profundas en estilos, estructura y componentes para ajustarla a la identidad visual de Saludy.

## Resultado
El sitio final entregado fue:

- Ligero, rápido y responsive.
- Fácil de navegar para usuarios de todas las edades.
- Visualmente consistente con la marca.
- Preparado para futuras mejoras o rediseños (incluyendo una posible migración a Astro).

El proyecto cumplió con los requerimientos del cliente y dejó una base sólida para evoluciones posteriores.
